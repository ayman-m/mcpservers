Cortex XQL Schema Reference Guide


1 Introduction
==============

This guide describes the fields available in the xdr\_data dataset. This dataset contains all data collected from all Cortex product agents, including EDR data, and PAN NGFW data.

2 XDR\_DATA Fields by Actor
===========================

This section lists the xdr\_data dataset fields grouped according to each actor.
